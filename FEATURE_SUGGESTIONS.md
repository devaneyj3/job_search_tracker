# Job Tracker - Feature Suggestions

## Current Features Summary
- Job application tracking (create, view, update, delete)
- Status tracking (Applied, Interview, Offer, Rejected, Archived)
- Contact information and email follow-ups
- Connection/outreach management
- Basic metrics dashboard
- Calendar event generation (.ics files)
- Email sending functionality
